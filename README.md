#EQUINOX.js
---
##Setup
1. Run **scan.js** to automatically generate a list of lights
2. Use **blink.js** to find lights by ip
3. Give lights custom names in lights.json (this is required!)
a. example: 
```json
[
    {
        "address":"0.0.0.0",
        "id":"asouedfhgdisakjfb",
        "model":"324-djafh",
        
        "name":"lights"
    }
]
```
